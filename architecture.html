Robotruck MPC Control Architecture

TruckSim — Simulink — ROS — Estimation — MPC · 30 Hz closed-loop

Control Layer

  • • PID longitudinal: v_ref = 20 m/s → throttle/brake
  • • MPC lateral: N=50, dt=1/30s, 4-state bicycle model
  • • Discrete A/B matrices: 3-axle stiffness (kf, km, kr)
  • • Veh params updated every 10s from estimation
  • • Publishes /guardian/control_cmd (th, br, δ)

Estimation Layer

  • • mass_slope_est: 3-state EKF → m ∈ [61t,152t], slope(tan)
  • • veh_param_est: UKF(tire forces) → EKF(Cf,Cr,Cxf,Iz)
  • • adhesion_est: 6-state UKF + Dugoff tire → μ×6
  • • Dual work_mode: empty (61t) / loaded (152t)
  • • All nodes: 30 Hz, /vcu + /localization subscriber

Simulation Environment

  • • TruckSim 2019: 8-DOF vehicle dynamics (6-wheel)
  • • Simulink 2020a: model bridge + ROS I/O
  • • Windows ↔ Ubuntu dual-PC via TCP
  • • ROS Master on Ubuntu, rosinit from MATLAB
  • • Custom msgs: robominer_msgs + estimation
